🤖 CAMADA 2 — TRIAGEM AUTOMÁTICA (IA SIMULADA, NÃO MOCK)

Criar um módulo de triagem inteligente, baseado em regras, mas estruturado como “IA”.

Fluxo:
	1.	Usuário abre chat
	2.	Sistema inicia bot de triagem
	3.	Bot pergunta o motivo do contato
	4.	Classifica:
	•	departmentId
	•	priority
	•	tags (ex: withdrawal, kyc, bug)
	5.	Se possível, responde automaticamente
	6.	Se não resolver → entra na fila humana

⚠️ NÃO usar respostas hardcoded no frontend
⚠️ Toda lógica fica no backend (support_triage_service)
