🧩 PARTE 1 — KYC LEVE (OBRIGATÓRIO)
📌 Requisitos de KYC

Implementar KYC básico com os seguintes campos:

Nome completo

CPF

Data de nascimento (opcional, mas recomendado)

📌 Regras de validação

CPF:

Validar formato

Validar dígitos verificadores (algoritmo oficial)

CPF deve ser único por conta

Nome:

Obrigatório

Salvo exatamente como informado pelo usuário

Status KYC:

PENDING

VERIFIED

REJECTED

📌 Comportamento

Usuário PODE depositar sem KYC

Usuário NÃO pode sacar sem KYC aprovado

Se tentar sacar sem KYC:

Mostrar modal:
“Para realizar um saque, é necessário validar seus dados (CPF).”
